Ejercicio 2

Galería de Tarjetas

Diseño Web

Interfaces modernas y atractivas.

El diseño web combina principios visuales con usabilidad. Incluye tipografía, color, espaciado y jerarquía visual. Herramientas más usadas: Figma, Adobe XD y Canva.

Desarrollo

Front-end y back-end actuales.

El desarrollo web se divide en frontend (HTML, CSS, JS) y backend (servidores y bases de datos). Frameworks populares: React, Vue, Node.js y Laravel.

Responsive

Adaptable a cualquier pantalla.

Usa media queries, Flexbox y CSS Grid para que tu sitio se vea bien en cualquier dispositivo. El principio base es "mobile-first": diseñar primero para móvil.

Rendimiento

Velocidad y tiempos de carga.

Un sitio rápido mejora la experiencia y el SEO. Técnicas: imágenes en formato WebP, lazy loading, minificación de CSS/JS y uso de CDN para archivos estáticos.

Seguridad

Ciberseguridad para sitios web.

Protege tu sitio con HTTPS, validación de formularios y sanitización de datos. Debes conocer y prevenir ataques como XSS, CSRF e inyección SQL.

SEO

Posicionamiento en buscadores.

Mejora tu visibilidad en Google con etiquetas correctas, URLs amigables, contenido de calidad y tiempos de carga rápidos. Es clave para atraer tráfico orgánico.

E-commerce

Tiendas en línea funcionales.

Las tiendas online incluyen catálogo, carrito y pasarelas de pago como Stripe o PayPal. Plataformas populares: WooCommerce (WordPress) y Shopify.

Analytics

Métricas del comportamiento.

Con Google Analytics y Hotjar puedes ver cuántas personas visitan tu sitio, qué páginas ven más y desde qué país o dispositivo acceden.
